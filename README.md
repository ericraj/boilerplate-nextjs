## Getting Started

- **Setup package manager and install dependencies**

```bash
npm install -g pnpm
pnpm install
```

- **Run the development server**

```bash
pnpm dev
```

## Techno

- **[Next.js App Router](https://nextjs.org/docs)**
- **[Shadcn UI](https://ui.shadcn.com/docs)**
- **[Lucide icons](https://lucide.dev/icons)**
